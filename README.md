# quizbot

Небольшой Telegram-бот на `aiogram 3` с квизом и сохранением последнего результата в `SQLite`.

## Что умеет

- Вопросы с вариантами ответов (inline-кнопки)
- После нажатия кнопки клавиатура убирается, в чат выводится выбранный вариант
- Подсчёт правильных ответов
- Сохранение **последнего** результата прохождения
- Команда `/stats` показывает последний результат

## Команды

- `/start` — стартовое сообщение + кнопка «Начать игру»
- `/quiz` или «Начать игру» — начать квиз
- `/stats` — последний результат

## Запуск

Требуется Python 3.11+.

1) Установить зависимости:

```bash
pip install -r requirements.txt
```

2) Создать `.env` рядом с `main.py`:

```env
BOT_TOKEN=ваш_токен_бота
```

3) Запустить:

```bash
python main.py
```

## Где что лежит

- `main.py` — запуск бота и подключение роутеров
- `handlers.py` — команды `/start`, `/quiz`, `/stats`
- `keyboards.py` — генерация inline-кнопок и обработка ответов
- `quiz_logic.py` — список вопросов/вариантов
- `db.py` — SQLite-таблицы и функции для состояния/результатов

Файл базы `quiz_bot.db` создаётся автоматически при первом запуске.
